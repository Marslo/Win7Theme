Win7Theme
=========
Theme for win7

## Marslo1.theme
<pre><code>Add "marslo1.theme" to %LOCALAPPDATA%\Microsoft\Windows\Themes
</code></pre>

## marslo-theme
<pre><code>Copy Win7THeme\ThemeFile\Marslo\* to %USERPROFILE%\AppData\Local\Microsoft\Windows\Themes\
</code></pre>
